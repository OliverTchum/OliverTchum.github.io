Live Chatrooom App - BuzzRooms

This is the Personal Project I'm currently working on callled BuzzRooms.It is a Full-Stack real-time chat application designed for seamless and dynamic communication in various rooms. Built to support both small group interactions and large communities, it allows users to create, join, and manage chat rooms. It was made with the goal of letting groups of strangers meet to talk about common intrests. In making this ive been developing my skills in full stack develeopment utilising react.js for the frontend and firebase cloud services such as the Firestore database and Firebase Authentication for the backend.
